# Shadesblue — Jason Martin

Senior Front-End/Web Engineer portfolio. Built with Next.js (App Router) + Tailwind. Focused on accessibility (WCAG 2.2 AA), performance (Core Web Vitals), and maintainability. Includes case-study pages with Problem / What I did / Impact / Training / Stack and a lightbox gallery.

- **Live site:** https://www.shadesblue.com/
- **Contact:** [Email](mailto:you@example.com) • Memphis, TN

---

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **UI:** React 19, Tailwind CSS
- **Images/Perf:** Next/Image, WebP, responsive `sizes`
- **Accessibility:** ARIA, focus management, keyboard nav, color contrast
- **Testing:** Vitest + React Testing Library (`jest-dom`)
- **TypeScript:** Strict mode
